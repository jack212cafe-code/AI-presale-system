Create lib/chat.js exporting a single function `handleChatMessage(params)`.

**Function signature:**
```javascript
export async function handleChatMessage({ conversationId, message, userId })
```

Returns: `{ conversation_id, project_id, stage, text, created }`
- `created`: boolean, true only when a new conversation was created on this call
- `text`: string, the assistant's markdown response
- `stage`: string, the conversation's stage after processing

**State machine logic (per D-01, D-02, D-03):**

Stage transitions:
- `greeting` (or no conversationId): First message. Auto-advance through intake + discovery.
  1. Call `normalizeIntakePayload({ customer_name: "Chat Project", primary_use_case: message, notes: message })` to create a minimal intake from the user's free-text brief
  2. Call `createProjectRecord(intake, userId)` to create project
  3. Call `createConversation(projectId, userId)` to create conversation
  4. Call `addMessage(conversationId, 'user', message)` to persist user message
  5. Call `runDiscoveryAgent(intake, { projectId })` for requirements
  6. Call `persistRequirementsJson(projectId, requirements)`
  7. Auto-advance: call `runSolutionAgent(requirements, { projectId })`
  8. Call `persistSolutionJson(projectId, solution)`
  9. Call `updateConversationStage(conversationId, 'awaiting_selection')`
  10. Format solution as markdown text showing numbered options
  11. Call `addMessage(conversationId, 'assistant', responseText)`
  12. Return `{ conversation_id, project_id, stage: 'awaiting_selection', text: responseText, created: true }`

- `awaiting_selection`: User picks a solution option (per D-02, D-14).
  1. Call `addMessage(conversationId, 'user', message)`
  2. Parse selection from message: look for a number (1, 2, 3...) in the message text using regex `/(\d+)/`. Extract first match as `selectedIndex`. Default to 1 if no number found.
  3. Load project via conversation.project_id: `getProjectById(conversation.project_id)`
  4. Extract the selected option from `project.solution_json` (array index selectedIndex - 1, clamped to valid range)
  5. Call `runBomAgent(selectedOption, { projectId: conversation.project_id, requirements: project.requirements_json })`
  6. Call `persistBomJson(conversation.project_id, bom)`
  7. Call `updateConversationStage(conversationId, 'bom')`
  8. Auto-advance: call `approveProject(conversation.project_id)` (per Phase 1 D-03: gate disabled)
  9. Call `runProposalAgent(project.intake_json, project.requirements_json, selectedOption, bom, { projectId: conversation.project_id })`
  10. Call `updateConversationStage(conversationId, 'complete')`
  11. Format BOM as markdown table and proposal download info as markdown text
  12. Call `addMessage(conversationId, 'assistant', responseText)`
  13. Return `{ conversation_id, project_id: conversation.project_id, stage: 'complete', text: responseText, created: false }`

- `complete`: Conversation finished. Return a message saying the proposal is ready and they can start a new conversation.

- Any other stage or error: Return `{ conversation_id, project_id, stage, text: "An error occurred...", created: false }`

**Handling new vs existing conversation:**
- If `conversationId` is null/undefined: treat as new conversation (greeting stage)
- If `conversationId` is provided: call `getConversationById(conversationId)` to load current stage, then dispatch to the appropriate stage handler

**Error handling:**
Wrap each stage in try/catch. On agent failure:
- Call `addMessage(conversationId, 'assistant', errorText)` with a user-friendly message like `"Sorry, an error occurred during [stage_name]: [error.message]. Please try again."`
- Do NOT change conversation stage on error (allows retry)
- Return the error text as the response

```javascript
if (request.method === "POST" && url.pathname === "/api/chat") {
  if (!requireUserAuth(request, response)) return;
  try {
    const payload = await parseBody(request);
    const userId = getSessionUserId(request);
    const result = await handleChatMessage({
      conversationId: payload.conversation_id || null,
      message: payload.message,
      userId
    });

    return json(response, result.created ? 201 : 200, {
      ok: true,
      conversation_id: result.conversation_id,
      project_id: result.project_id,
      stage: result.stage,
      text: result.text
    });
  } catch (error) {
    return json(response, 500, { ok: false, error: error.message });
  }
}
```
