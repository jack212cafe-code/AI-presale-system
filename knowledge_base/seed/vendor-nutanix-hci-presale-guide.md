# Vendor Nutanix HCI Presale Guide

- Position Nutanix when the customer values operational simplicity, lifecycle management consistency, fast cluster expansion, and strong virtualization integration over deep component-level customization.
- Validate the cluster design against node failure tolerance, maintenance windows, and storage rebuild behavior. A correct presale design explains what happens during patching and host loss, not only steady-state resource totals.
- Separate hypervisor, management, and data protection assumptions. Backup immutability, DR replication, and application-consistent recovery remain separate design decisions even when the platform is operationally unified.
- Be explicit about licensing scope, feature edition, hypervisor choice, and external dependencies such as backup targets, directory services, DNS, and monitoring.
- For customers moving from three-tier virtualization, explain operational tradeoffs clearly: simpler lifecycle and scaling versus less freedom for highly specialized storage tuning.
