# Questions DevOps Discovery Framework

- How does code move from developer workstation to production today, and where are the biggest bottlenecks or failure points?
- What tools are currently used for source control, CI/CD, infrastructure automation, secrets, observability, and release approval?
- Which responsibilities belong to application teams today, and which ones should move into a shared platform model?
- How are environments promoted, rolled back, and audited, and where does the current process depend on manual steps?
- What compliance, security, and approval controls must be preserved inside the delivery flow?
- Which service reliability expectations matter most: deployment speed, change success rate, recovery speed, or operational consistency?
- Is the customer asking for a tooling refresh, a delivery model transformation, or both?
